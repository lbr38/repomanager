A scheduled task is basically a normal task that is executed at a defined date and time instead of being executed immediately.

The task can either be a single unique task executed once at a defined date and time, or a recurring task executed on a defined frequency (hourly, daily, weekly...).

## New scheduled task

From the **REPOS** tab:

1. Either create a new repository using the **+ Create a new repository** button or select an existing repository and click the task you want to execute.
2. Specify the task parameters, then click the **Schedule it** button to define the schedule parameters.
3. Click the **Schedule** button to confirm the schedule.

## Disable a recurring task

Recurring tasks can be temporarily disabled without being totally cancelled.

From the **TASKS** tab:

1. Use the Disable button to disable a recurring task.

From the **TASKS** tab:

1. Use the Delete icon to cancel a scheduled task.
