

## Repository statistics

To enable repository **statistics and metrics**, see **REPOSITORIES > Statistics >** `Enable repositories statistics <https://github.com/lbr38/repomanager/wiki/08.-Settings#statistics>`_ section from the **SETTINGS** tab.

From the **REPOSITORIES** tab:

1. Locate the repository you want to visualize statistics for.
2. Click on the ``Release version`` or ``Dist / component`` to access the statistics tab.
